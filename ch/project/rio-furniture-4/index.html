語音轉文字

將語音檔轉成文字

Project Details

  • 客戶 : 盧老師
  • 內容 : 語音檔轉文字
  • 工具 : Python
  • 技能 : Python

Google 的語音檔轉成文字


老師的需求

盧老師除了在學校教書之外，還持續在研究所進修。針對研究內容，請學生各自錄製自得，總計約有四十個音檔需要分析。一般的做法會請人聽打，曠日廢時，同一個音檔還需來回重複聽許多次，因此，使用 AI 技術將語音檔轉成文字，便成為了本專案首要目標。


設計架構

使用 Google 提供的語音辨識服務，將訪談音檔丟到 Google Cloud Service 的 Bucket，並使用 Python 呼叫雲端服務，將雲端上的音檔轉換成文字並輸出到自己的電腦。

展示影片可加入臉書「薩格茲學院」社團觀看。
