作文批改及自動收集


老師需求

學生的寫作每句之間總是不連貫，需要好好地引導每句該寫的內容和方向。另外，批改作文非常耗時，常常讓老師身心俱疲。


設計思維

使用 Python 和 Google Colab 一起收集所有學生上傳的檔案，計算字數、用字、Token Ratio等等。同時利用 input() 和 Loop，用引導式問題，讓學生寫出連貫性高的作文。
