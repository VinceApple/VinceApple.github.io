LINE & Pandorabot

Building a chatbot on LINE

Project Details

  • What We Did : Chatting Robot
  • Tools Used : LINE, Python, Pandorabot
  • Skills : Python / AIML / Git

General-Purpose Chatbot Design


The project aims to leverage the open-source AIML files to build a chatbot on LINE. EFL teachers can customize the chatbot’s properties. They can also add some phrases they would like to teach to the chatbot’s knowledge bank.
